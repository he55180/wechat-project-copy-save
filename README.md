# Wechat Project Copy-Save

> 微信公众号文章一键抓取 · 自动排版 · 归档系统

## 🔥 核心工具: Wechat Project Copy-Save (一键文章归档)·Word版)

**功能**：复制微信链接 -> 自动转存为 Markdown 和 **Word (.docx)** 格式（自动套用公文模板）。

### 📂 核心文件

| 文件名 | 说明 |
| :--- | :--- |
| **`一键保存.bat`** | **[入口]** 双击运行。自动检测环境并启动任务。 |
| `clip_save.py` | **[逻辑]** 包含剪贴板读取、网页解析、图片修复、Pandoc转换逻辑。 |
| `黄金模板-总结报告...docx` | **[模板]** Pandoc 转换时引用的参考文档格式。 |
| `requirements.txt` | **[依赖]** Python 依赖库清单 (requests, html2text, pyperclip等)。 |
| `.env` | **[配置]** 环境变量配置文件。 |

### 🛠️ 使用方法

1. 在电脑微信上打开公众号文章。
2. **右键**点击内容区域 -> 选择 **“复制地址”** (或右上角复制链接)。
3. 双击 **`一键保存.bat`**。
4. 等待约 5-10 秒，提示“✅ Word 保存成功”。
5. 文件默认保存在：`C:\HSE_Knowledge_Base`。

## 🔧 常见问题

* **Word 转换失败？** 请确保安装了 `pandoc`，且同目录下存在`黄金模板...docx`文件。
* **图片不显示？** Word 文档中的图片已做防盗链处理，通常能正常显示。如果网络不通，请检查网络连接。
* **报错闪退？** 脚本已包含自动安装依赖功能，首次运行请确保联网。
