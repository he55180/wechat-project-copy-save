# Wechat Project Copy-Save

> 微信公众号文章一键抓取 · 自动排版 · 归档系统

## 🔥 核心工具: Wechat Project Copy-Save (一键文章归档·Word版)

**功能**：复制微信链接 -> 自动转存为 Markdown 和 **Word (.docx)** 格式（自动套用公文模板 + GB/T 9704-2012 排版引擎）。

提供两种操作模式：**GUI 图形界面**（推荐）和 **CLI 命令行**（轻量快捷）。

---

### 🖥️ 模式一：GUI 图形界面（推荐）

基于 tkinter 的桌面客户端，内置实时日志、设置面板、Pandoc 转换与 GB/T 9704-2012 排版管线。

| 文件名 | 说明 |
| :--- | :--- |
| **`启动归档系统GUI.bat`** | **[入口]** 双击启动 GUI 窗口。自动检测 Python 环境。 |
| `app.py` | **[逻辑]** tkinter GUI 客户端（554行），含剪贴板监控、实时日志、路径配置、一键归档。 |
| `format_expert.py` | **[排版]** GB/T 9704-2012 公文排版引擎，自动规范化标题层级、正文字体、段落间距。 |
| `custom_settings.json` | **[配置]** GUI 用户偏好设置（保存目录等）。 |

#### 使用方法（GUI）

1. 在电脑微信上打开公众号文章，复制链接。
2. 双击 **`启动归档系统GUI.bat`**。
3. 在 GUI 窗口中点击 **”一键保存”**。
4. 右侧日志区实时显示处理进度。
5. 点击右上角 **⚙ 设置** 可修改保存目录。

---

### ⌨️ 模式二：CLI 命令行（轻量快捷）

| 文件名 | 说明 |
| :--- | :--- |
| **`一键保存.bat`** | **[入口]** 双击运行。自动检测环境并启动任务。 |
| `clip_save.py` | **[逻辑]** 包含剪贴板读取、网页解析、图片修复、Pandoc 转换逻辑。 |

#### 使用方法（CLI）

1. 在电脑微信上打开公众号文章。
2. **右键**点击内容区域 -> 选择 **”复制地址”** (或右上角复制链接)。
3. 双击 **`一键保存.bat`**。
4. 等待约 5-10 秒，提示”✅ Word 保存成功”。
5. 文件默认保存在：`C:\HSE_Knowledge_Base`。

## 🔧 常见问题

* **Word 转换失败？** 请确保安装了 `pandoc`，且同目录下存在`黄金模板...docx`文件。
* **图片不显示？** Word 文档中的图片已做防盗链处理，通常能正常显示。如果网络不通，请检查网络连接。
* **报错闪退？** 脚本已包含自动安装依赖功能，首次运行请确保联网。
